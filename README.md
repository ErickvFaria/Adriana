# 👨🏽‍💻 Disciplina: Linguagem de Programação  

Repositório destinado às atividades e exercícios da disciplina de **Linguagem de Programação**, ministrada pela professora **Adriana da Silva Jacinto** na **Faculdade de Tecnologia de São José dos Campos – Professor Jessen Vidal**.  

---

## 📚 Atividades Desenvolvidas  

### 📊 1. Teste de Mesa  
- Aplicação da fórmula de cálculo da média considerando:  
  - Provas (**P1, SUB**)  
  - Entregas (**E1, E2**)  
  - Atividade Extra (**X**)  
  - Projeto Integrador (**API**)  
- Elaboração de **10 cenários** diferentes, como:  
  - Aluno sem nota na P1 mas com boas notas nas demais.  
  - Aluno com ótimo desempenho em todas as avaliações.  

---

### 🎵 2. Modelagem Conceitual a partir de uma Música  
- Seleção de uma música de preferência.  
- Criação de **3 classes** (com 3 atributos e 1 método cada).  
- Representação em **diagrama UML**.  
- Cada classe deve estar relacionada a elementos presentes na letra da música.  

---

### 🎨 3. Modelagem Conceitual inspirada em Vik Muniz  
- Escolha de **duas obras do artista**.  
- Para cada obra:  
  - Desenvolvimento de **3 classes** (3 atributos e 1 método cada).  
  - Representação em UML dos objetos identificados nas imagens.  

---

### ✏️ 4. Modelagem Conceitual a partir de um Doodle  
- Seleção de uma ilustração (ex.: [Mauro Martins](https://www.mauromartins.com/)).  
- Estruturação de **3 classes** (3 atributos e 3 métodos cada).  
- Modelagem em diagrama UML dos objetos representados no desenho.  

---

### 🎬 5. Modelagem Conceitual baseada em um Vídeo  
- Escolha de um vídeo de interesse.  
- Criação de **3 classes** (3 atributos e 3 métodos cada).  
- Representação em UML dos objetos visualizados no vídeo.  

---

### ☁️ 6. Publicação no GitHub  
- Upload de todos os **diagramas UML** e materiais utilizados nos exercícios 2 a 5.  

---

### 💻 7. Implementação em Java – Exercício 1  
- Codificação em **Java** da classe que realiza o **cálculo da média** do Teste de Mesa.  
- Entrada de dados via teclado e saída do resultado final.  

---

### 🔧 8. Implementação em Java – Exercícios 2 a 5  
- Desenvolvimento das classes geradas a partir das modelagens:  
  - Música 🎵  
  - Obras de Arte 🎨  
  - Doodle ✏️  
  - Vídeo 🎬  

---

### 📂 9. Publicação no GitHub  
- Upload de todo o **código Java** implementado nos exercícios anteriores.  

---

## 🎯 Objetivo Geral  
Consolidar os conceitos de **modelagem orientada a objetos** e aplicá-los na **programação em Java**, passando pelas etapas de:  
1. Análise  
2. Design (modelagem UML)  
3. Implementação (código Java)  
4. Publicação no GitHub
